Collection of random words I spent way too long scraping from websites. I didn't find any random word API that was just a request. You can get the source and host it yourself if you want. It is now running on a hobby dyno, so it should be up quite reliably. Runs under the DWTFYW license.

Send a GET request to
(where number = number of words returned.)

/word?number=10

If you want all the stored words:

/all

You can toggle swear words on/off (0/1) with the filter

/?swear=0
